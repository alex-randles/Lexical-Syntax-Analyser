# Lexical & Syntax Analyser

This assignment uses javacc to create a parser. The aim of this assignment is to implement a lexical and syntax analyser using JavaCC for a simple language called CAL.
The details of the CAL language are given https://github.com/randlea2/LexicalSyntaxAnalyser/blob/master/cal.pdf
